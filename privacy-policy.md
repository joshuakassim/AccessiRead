# AccessiRead Privacy Policy

**Last updated:** November 2025  

AccessiRead is a Chrome browser extension that enhances the readability of web pages for users with dyslexia, ADHD, and other reading difficulties.  

---

## Data Collection

AccessiRead **does not collect, store, or transmit any personal or identifiable information**.  
All user preferences (such as font type, size, color, or layout settings) are stored **locally** in the user’s browser using Chrome’s `storage` API.  

---

## Data Sharing

AccessiRead **does not share, sell, or transmit** any information to external servers, advertisers, or third parties.  

---

## Permissions

The extension requests the following Chrome permissions:

- **`activeTab`** and **`scripting`** – to modify webpage styling only when AccessiRead is active.  
- **`storage`** – to save your preferences locally.  
- **`tabs`** and **`host_permissions (""https://*/*", "http://*/*"")`** – to apply readability features on any site you choose.  

These permissions are used solely for core functionality and never for data collection.
